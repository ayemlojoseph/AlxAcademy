First is to initiate a git repo

command:
        git init
this will make a folder a git repo

secondly go to github and create a repo

clone the repo
command:        
        git clone repoName
the repo name can be copied from the github like git@github.com:ayemlojoseph/AlxAcademy.git


Addding a single file to the git repo >>>>>>>
command:
        git add example.txt

adding all files to the git repo>>>>>>>>>
command:
        git add .

commiting files>>>>>>>>>
command:
        git commit -m "Initial commit"

pushing to the repo>>>>>>>>>>>>
command:
        git push origin main

checking status of files
command:
        git status
